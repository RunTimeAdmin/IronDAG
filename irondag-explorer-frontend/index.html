Deployment Guide

Compile and deploy Solidity contracts

Option A — Remix IDE (easiest)
Open remix.ethereum.org, write or import your contract, compile it, then set Environment to Injected Provider — MetaMask. IronDAG will appear as the connected network. Click Deploy.
Option B — Hardhat
npx hardhat compile
Find ABI and bytecode in artifacts/contracts/YourContract.sol/YourContract.json. Paste bytecode (the "bytecode" field) and ABI array here.
Hardhat network config
irondag: { url: "https://explorer.irondag.io/rpc", chainId: 11567, accounts: [process.env.PRIVATE_KEY] }
Connect your wallet to call write functions. Read functions work without a wallet.
