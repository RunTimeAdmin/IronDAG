========================================
  IronDAG Blockchain Node
  Quick Start Guide
========================================

INSTALLATION:
1. Extract this ZIP file to any folder
2. Double-click "start_node.bat" to start the node
3. Wait for "RPC server listening on 127.0.0.1:8546"

USING THE DESKTOP APP:
1. Download and install the IronDAG Desktop app
2. Open the desktop app
3. It will automatically connect to the node

USING METAMASK:
1. Open MetaMask
2. Add Custom Network:
   - Network Name: IronDAG Testnet
   - RPC URL: http://127.0.0.1:8546
   - Chain ID: 11567
   - Currency Symbol: IDAG
3. Connect and start using!

TROUBLESHOOTING:
- If port 8546 is in use, close other applications using that port
- Make sure Windows Firewall allows the connection
- Check that node.exe is in the same folder as start_node.bat
